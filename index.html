About Me

Full stack Electronics Engineer with 12 years of experience in the Hardware, Firmware & Software domains. I like to work with low-level systems that directly interact with silicon.

I have been involved in multiple projects from the ground up and took it up to the production-ready firmware and DFM of PCBA. including EMI certifications.

More About Me

I started building my circuits in the Analog domain in my teenage. Then I picked up digital Electronics and then got introduced to the Instruction set abstraction layer. This is where I started coding with plain assembly and understanding the computing architecture. RAM, ROM, Timers, Interrupts, WDT, etc. From there I have been going through all sorts of programming languages in this order -> C, C++, Objective-C, Python, Swift, and Rust.

I love designing my own hardware, laying out the PCB, sourcing the components, assembling the board and bringing it into life by writing the very first line of code which it will execute once it is powered on and up to the stage where it does over-the-air firmware updates or runs some ML model inference at the edge

I also love developing mobile Apps. I am highly skilled in Native iOS mobile Applications development and Developing cross-platform Apps using Flutter. Since I started Flutter in 2019, Flutter has been my first choice for any mobile app development requirement

My path to where my knowledge is now can be mapped as below Passive Components and LEDs -> Transistors -> 555 Timers -> OpAmps -> Logic Gates -> FPGA -> ISA -> Assemblers -> Compilers -> OS -> Runtime environments -> Modern programming languages
